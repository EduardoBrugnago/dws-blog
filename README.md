# DWS Blog

A blog application built with React and Vite.

## Tech Stack

- **React 19**
- **Vite**
- **Styled Components** 
- **Redux Toolkit** 
- **React Router** 
- **Axios**

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens the app in development mode with hot module replacement.

### Build

```bash
npm run build
```

Generates an optimized production build in the `dist/` folder.
